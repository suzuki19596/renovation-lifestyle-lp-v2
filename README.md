# リノベーション不動産LP v2

## 概要

30代女性をターゲットにした中古×リノベーション不動産のランディングページ（第2案）です。
「普通の部屋で、普通の人生を終わらせない」をコンセプトに、自分らしさを取り戻すためのリノベーションを訴求します。

## コンセプト

### ターゲット
- **メインターゲット**: 30代一人暮らしの女性
- **ペルソナ**: 仕事は頑張っているが、家に帰っても自分らしさを感じられない女性
- **ニーズ**: 自分の世界観を表現できる、心地よい空間づくり

### デザインコンセプト
- **テーマカラー**: ホワイト × くすみグリーン / ベージュピンク
- **雰囲気**: スタイリッシュ・余白・洗練・"自分時間"
- **写真テイスト**: カフェ・北欧・ナチュラルモダン

## ページ構成

### 1. ファーストビュー
- **H1**: 「普通の部屋で、普通の人生を終わらせない。」
- **H2**: 自分らしさを取り戻すためのリノベーション。"住まい"を、自分の人生の舞台に変える。
- **CTAボタン**: 「理想の空間を無料シミュレーション」

### 2. 課題共感セクション
- **H2**: 「好き」が反映されない空間に、もう疲れた。
- ターゲットの痛みに共感する3つの画像とメッセージ
- 退屈な部屋、無機質なインテリア、疲れた帰宅シーン

### 3. 便益・差別化セクション
- **H2**: "中古＋リノベ"で、世界にひとつだけの空間を。
- 3つの便益を紹介:
  1. 限られた予算で、理想のデザインと機能を両立
  2. 空間デザイナーと一緒に、自分の世界観を形に
  3. 新築ではできない「私仕様」の自由設計

### 4. 感情訴求セクション
- **H2**: "おしゃれな部屋"じゃない。"私が生き返る場所"。
- 自分らしい空間が人生を変えるというメッセージ
- 女性が笑顔でソファに座るビジュアル

### 5. 施工事例セクション
- **H2**: "好き"を詰め込んだ、世界にひとつの部屋。
- 3つの事例を紹介:
  1. 北欧カフェ風1LDK
  2. コンクリート×グリーンのモダン空間
  3. ミニマル×機能美の1LDK

### 6. お客様の声セクション
- **H2**: 自分の部屋が、人生を変えた。
- 2名のお客様の声:
  1. 「リノベ後は、毎日が"好き"で埋め尽くされてる。」
  2. 「部屋にいる時間が、自分のご褒美になった。」

### 7. ファイナルCTA / お問い合わせフォーム
- **H2**: "部屋を変える"は、"人生を取り戻す"ということ。
- 完成後の部屋と笑顔の女性のビジュアル
- お問い合わせフォーム（名前、メール、電話、相談内容、メッセージ）
- **CTAボタン**: 「理想の部屋を無料相談」
- **補足文**: オンラインOK／女性プランナー在籍

## 使用方法

1. プロジェクトフォルダをWebサーバーに配置
2. `images/`フォルダに必要な画像を配置
3. `index.html`をブラウザで開く

## 注意事項

- 画像は軽量化（WebP推奨）して配置してください
- フォームの送信機能は実装されていません（実際の運用時は backend と連携）
- プライバシーポリシーのリンク先を実際のURLに変更してください

## カスタマイズポイント

### カラー変更
`css/style.css`の`:root`セクションでカラー変数を変更できます。

### フォント変更
HTMLの`<head>`セクションとCSSの`--font-display`、`--font-sans`を変更してください。

### スペーシング調整
`:root`セクションの`--spacing-*`変数を調整することで、全体の余白を変更できます。
